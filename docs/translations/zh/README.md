# 2048

逐步教程，引导读者使用 Bevy 引擎实现 2048 项目。

## 步骤

为了使教程更容易上手，我们将整体教程分为四个「步骤」和五个「拓展」。
「核心步骤」旨在实现经典款2048游戏的功能。「功能」

本教程为初级教程，并不采用数据优先的设计模式。而是先构建简单图形界面。
